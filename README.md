# Moon Runner

A pixel art lunar rover side-scrolling adventure game built with vanilla HTML5 Canvas.

## Play

Open `index.html` in any modern browser, or play online via GitHub Pages.

## Controls

- **Tap / Click / Space / Arrow Up** — Jump
- **Arrow Left / Right** — Navigate rover selection
- **Space** — Confirm selection

## Features

- Lunar gravity physics (slow float up, slow fall down)
- 5 stages based on real lunar locations (Mare Tranquillitatis → South Pole Aitken)
- 3 unlockable rovers: Sojourner, Curiosity, Perseverance
- Obstacles: rocks, water pools, craters
- Energy crystals for booster refill
- Double-jump booster system
- Parallax scrolling backgrounds with planets, Earth, and Moon
- Random moon facts on game over
- PC and mobile compatible

## Tech

Single-file HTML5 Canvas game. No dependencies. Internal resolution 320×214, rendered at 3× scale (960×642).
